md2html-vim

Remote plugin for neovim that generates .html from .md file on save in same directory.

Dependencies

  • node.js
  • evilstreak/markdown-js (global installation)

Installation

Follow instructions for your plugin manager of you choice and then type :UpdateRemotePlugins

vim-plug

Plug 'lorentzlasson/md2html-vim'
